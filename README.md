# Atlas Athletique

Webapp statique pour suivre un systeme de developpement athletique sur 12 mois.

## Publication GitHub Pages

1. Cree un depot GitHub.
2. Place tous les fichiers de ce dossier a la racine du depot.
3. Dans GitHub: Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: `main`, folder: `/root`.
6. Ouvre l'URL GitHub Pages fournie.

## Installation iPhone

1. Ouvre l'URL GitHub Pages dans Safari.
2. Appuie sur Partager.
3. Choisis Ajouter a l'ecran d'accueil.

Les donnees de journal et de tests sont stockees localement dans le navigateur. Le bouton Exporter permet de sauvegarder un fichier JSON.
